# Market Wave Surface

This project visualizes and predicts financial market waves using historical price data for multiple assets. It features:
- 3D animated, liquid-style surface plots of asset returns over time
- Machine learning regression for next-step return prediction
- Neon/dark-themed, interactive visualizations with glow effects
- Modular, clean, and fully commented Python code

## What is this project about?
Market Wave Surface is a Python project that loads historical price data (from yfinance or sample data), computes returns, and visualizes them as a dynamic 3D surface. It overlays machine learning predictions and provides additional 2D/heatmap views. The project is designed for both financial analysis and stunning data art.

## How to fork and run
1. **Fork this repository** on GitHub (or download the code).
2. Clone your fork:  
   `git clone https://github.com/YOUR_USERNAME/Market-Wave-Surface.git`
3. Install requirements:
   `pip install -r requirements.txt`
4. Run the main script:
   `python main.py`
